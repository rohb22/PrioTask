# PrioTask

Android app built for one purpose:

**Open the app. Know what matters. Make progress. Close the app.**

PrioTask is intentionally small, fast, and opinionated. It does not try to be a project management suite. It focuses on helping you identify the next best task and move it forward.

## Goal

Most task managers create more work.

PrioTask removes decisions.

It's designed around a single question:

"""What should I do next?"""

The app uses urgency, priority, difficulty, progress, and task momentum to surface the most valuable work at any given moment.

No clutter. No endless customization.

Just clear priorities.
